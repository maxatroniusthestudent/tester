# FinanceFlow (HTML/CSS/JS)

## Запуск
1) Відкрий `index.html` подвійним кліком (або через Live Server у VS Code).
2) Дані зберігаються у `localStorage`.

## Loader (екран завантаження)
- Показується при першому заході.
- Показується також при оновленні сторінки (F5 / Ctrl+R) — через `sessionStorage` прапорець `ff_show_loader`.
